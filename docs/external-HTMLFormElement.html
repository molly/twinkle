getChecked(name[, type]) → Array of string

Get checked items in the form.

Parameters

Name Type Optional Description

name

string

 

Find checked property of elements (i.e. a checkbox or a radiobutton) with the given name, or select options that have selected set to true (don't try to mix selects with radio/checkboxes).

type

string

Yes

Optionally specify either radio or checkbox (for the event that both checkboxes and radiobuttons have the same name).
Returns

Array of string 

  • Contains the values of elements with the given name checked property set to true.
